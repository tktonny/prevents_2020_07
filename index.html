关于我们

团队介绍

  来自于上海财经大学的三个信息人，在19年聚在一起准备用自己所学知识做些贡献。在韩潇老师的指导下，我们决定对传染病进行可视化与预测。在这之后爆发的新冠疫情，我们希望用自己做出的成果对疫情的防治贡献微薄之力。

项目历程

  自19年的下半年开始，团队自主学习知识，充实自身，提升能力，并在明确分工之后着手进行项目开发。

在疫情爆发之后团队加紧项目开发的进程：

  • 运用 python 进行数据挖掘，爬取数据将应用于模型的建立以及疫情预测；
  • 使用 pandas及sklearn 进行模型建立完成较为准确的疫情预测；
  • 基于 echarts 对预测结果及爬取数据进行可视化 直观展现在地图以及各类图表之上。

现如今我们仍在不断地更新优化，使功能更加完善页面更加美观。
